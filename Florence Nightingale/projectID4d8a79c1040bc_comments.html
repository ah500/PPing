Proofing and Formatting
Process

This is a Newcomers Only/Rapid Review project. Once P1 has completed, the book will move quickly into P2 as a Rapid Review project. As soon as P2 completes, each P1 will receive a computer-generated report of the P2 diffs generated.
Proofing

Marking hyphenated words, e.g., to-*day, round-*house or post-*office, is always acceptable. Please leave previously marked end-of-line uncertain hyphenation (-*) alone--I have automatic scripts to handle them during PPing, so the more the merrier.
Formatting

When chapters start with poems or quotes, please mark with /# or /* as usual, and put the 2 blank lines after the poem/quote.

Please use <f> to markup regular text within an italic phrase. For example:
This is an Italian poem.
<i>This is an <f>Italian</f> poem.</i>
